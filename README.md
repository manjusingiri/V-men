# V Men - Women's Fashion Shopping UI

V Men is a frontend e-commerce interface designed for women's fashion shopping. Built with HTML, CSS, and JavaScript, it provides a clean, elegant, and responsive user experience.

## Features
- 💜 Modern white & purple UI
- 🛍️ Women's fashion shopping interface
- 🔍 Live product search
- ❤️ Wishlist toggle
- 📱 Responsive mobile-first design
- 🎯 Scrollable category section
- 🖼️ Promotional banner
- ⭐ Interactive product cards
- 🧭 Bottom navigation bar

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

This project was created to practice frontend development and build a visually appealing e-commerce user interface.
